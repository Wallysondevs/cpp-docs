# std::error_condition::assign

```cpp
void assign( int val, const error_category& cat ) noexcept;  // (desde C++11)
```

Atribui conteúdo a uma condição de erro. Define o valor de erro como `val` e a categoria de erro como `cat`.

### Parâmetros

- **val** — valor de erro
- **cat** — categoria de erro

### Valor de retorno

(nenhum)
