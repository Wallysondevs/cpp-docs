# std::auto_ptr&lt;T&gt;::release

T* release() throw(); | | (obsoleto em C++11)
(removido em C++17)

Libera o ponteiro mantido. Após a chamada, *this mantém o ponteiro nulo.

### Parâmetros

(nenhum)

### Valor de retorno

get().
