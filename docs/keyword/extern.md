# Palavra-chave C++: extern

### Uso  
  
  * [ duração de armazenamento estática com especificador de ligação _externa_](<#/doc/language/storage_duration>)
  * especificação de [ligação de linguagem](<#/doc/language/language_linkage>)

  * declaração de instanciação explícita de template (ou "extern template") 
    * [para templates de classe](<#/doc/language/class_template>)
