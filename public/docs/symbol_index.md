# Índice de Símbolos std

Esta página tenta listar todos os símbolos disponíveis na [biblioteca padrão](<#/doc/standard_library>) no namespace std. Os símbolos são escritos da seguinte forma:

*   Nomes de funções com `()`.
*   Templates com `<>`.

Símbolos dos sub-namespaces de std (por exemplo, chrono) não estão listados aqui, mas os nomes dos namespaces (precedidos pelo ícone ▶) são os links para as páginas correspondentes.

[Símbolos de macro](<#/>), [símbolos apenas para exposição](<#/>) e [símbolos removidos da biblioteca padrão](<#/>) estão listados em páginas separadas.
