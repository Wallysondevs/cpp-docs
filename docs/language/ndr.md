# Nenhum Diagnóstico Requerido

"Nenhum diagnóstico requerido" indica que alguma fraseologia é malformada de acordo com as regras da linguagem, mas um compilador não precisa emitir nenhum diagnóstico ou mensagem de erro. Geralmente, a razão é que tentar detectar essas situações resultaria em tempos de compilação proibitivamente longos.

Se tal programa for executado, [o comportamento é indefinido](<#/doc/language/ub>).
