# Cabeçalho da biblioteca padrão &lt;cassert&gt;

Este cabeçalho estava originalmente na biblioteca padrão C como [`<assert.h>`](<#/>).

Este cabeçalho faz parte da biblioteca de [tratamento de erros](<#/doc/error>).

### Macros

---
[ assert](<#/doc/error/assert>) | aborta o programa se a condição especificada pelo usuário não for verdadeira. Pode ser desabilitado para builds de lançamento.
(macro de função)

### Sinopse
```
    #define assert(E) /* see description */
```
