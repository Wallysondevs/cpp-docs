# SIG_ERR

Definido no cabeçalho `[<csignal>](<#/doc/header/csignal>)`

```c
#define SIG_ERR /* implementation defined */
```

Um valor do tipo `void (*)(int)`. Quando retornado por [std::signal](<#/doc/utility/program/signal>), indica que ocorreu um erro.
