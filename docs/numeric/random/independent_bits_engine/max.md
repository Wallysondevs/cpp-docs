# std::independent_bits_engine&lt;Engine,W,UIntType&gt;::max

```cpp
static constexpr result_type max();  // (desde C++11)
```

  
Retorna o valor máximo potencialmente gerado pelo adaptador de engine. Este valor é igual a 2w - 1. 

### Parâmetros

(nenhum) 

### Valor de retorno

O valor máximo potencialmente gerado. 

### Complexidade

Constante.
