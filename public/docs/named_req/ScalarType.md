# Requisitos nomeados C++: ScalarType

Nota: o padrão não define um requisito nomeado com este nome. Esta é uma categoria de tipo definida pela linguagem central. É incluída aqui como um requisito nomeado apenas por consistência.

### Requisitos

Os seguintes tipos são coletivamente chamados de _tipos escalares_ :

  * [tipos aritméticos](<#/doc/language/type-id>)
  * [tipos de enumeração](<#/doc/language/enum>)
  * [tipos ponteiro](<#/doc/language/pointer>)
  * [tipos ponteiro para membro](<#/doc/language/pointer>)

  * [std::nullptr_t](<#/doc/types/nullptr_t>)

| (desde C++11)

  * versões cv-qualified desses tipos
