# std::independent_bits_engine&lt;Engine,W,UIntType&gt;::min

```cpp
static constexpr result_type min();  // (desde C++11)
```

  
Retorna o valor mínimo potencialmente gerado pelo adaptador de engine. Este valor é igual a 0u. 

### Parâmetros

(nenhum) 

### Valor de retorno

O valor mínimo potencialmente gerado. 

### Complexidade

Constante.
