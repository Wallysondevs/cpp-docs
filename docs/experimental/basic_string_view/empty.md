# std::experimental::basic_string_view&lt;CharT,Traits&gt;::empty

constexpr bool empty() const noexcept; |  |  (library fundamentals TS)  

  
Verifica se a view não possui caracteres, ou seja, se `size() == 0`. 

### Parâmetros

(nenhum) 

### Valor de retorno

true se a view estiver vazia, false caso contrário 

### Complexidade

Constante.
