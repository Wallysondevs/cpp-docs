# std::locale::~locale

~locale();

  
Destrutor não-virtual que decrementa as contagens de referência de todas as facets mantidas por *this. Aquelas facets cujas contagens de referência se tornam zero são deletadas.

### Valor de retorno

(nenhum)

### Exemplo

| Esta seção está incompleta  
Razão: nenhum exemplo
