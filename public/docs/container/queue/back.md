# std::queue&lt;T,Container&gt;::back

reference back();
const_reference back() const;

  
Retorna uma referência para o último elemento na queue. Este é o elemento mais recentemente inserido. Efetivamente chama c.back(). 

### Parâmetros

(nenhum) 

### Valor de retorno

Referência para o último elemento. 

### Complexidade

Constante.
