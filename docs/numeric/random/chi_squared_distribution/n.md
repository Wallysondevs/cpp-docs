# std::chi_squared_distribution&lt;RealType&gt;::n

RealType n() const; | | (desde C++11)

Retorna o parâmetro n com o qual a distribuição foi construída. Ele especifica os [graus de liberdade](<https://en.wikipedia.org/wiki/Degrees_of_freedom_\(statistics\)> "enwiki:Degrees of freedom \(statistics\)") da distribuição. O valor padrão é 1.0.

### Parâmetros

(nenhum)

### Valor de retorno

O parâmetro n da distribuição.
