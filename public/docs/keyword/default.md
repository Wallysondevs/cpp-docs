# Palavra-chave C++: default

### Uso

  * [`switch` statement](<#/doc/language/switch>): como a declaração do rótulo de caso default

  * definição de função explicitamente default: como uma instrução explícita para o compilador gerar uma [função membro especial](<#/doc/language/member_functions>) ou um [operador de comparação](<#/doc/language/default_comparisons>)(desde C++20) para uma classe.
