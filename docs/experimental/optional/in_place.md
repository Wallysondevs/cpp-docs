# std::experimental::in_place

Definido no cabeçalho `[<experimental/optional>](<#/doc/header/experimental/optional>)`

```c
constexpr in_place_t in_place{};
```

  
`std::experimental::in_place` é uma constante do tipo [std::experimental::in_place_t](<#/doc/experimental/optional/in_place_t>) que é usada para desambiguar as sobrecargas de construtores e funções membro que aceitam argumentos (possivelmente um parameter pack) para a construção in-place de algum valor.
