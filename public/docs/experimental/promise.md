# std::experimental::promise

`std::experimental::promise` pode se referir a:

  * [`std::experimental::fundamentals_v1::promise` ou `std::experimental::fundamentals_v2::promise`](<#/doc/experimental/lib_extensions/promise>), uma versão modificada de [std::promise](<#/doc/thread/promise>) fornecida pelo [library fundamentals TS](<#/doc/experimental/memory>) com suporte para alocadores type-erased;
  * [`std::experimental::concurrency_v1::promise`](<#/doc/experimental/concurrency/promise>), uma versão modificada de `std::promise` fornecida pelo [concurrency TS](<#/doc/experimental/concurrency>) cuja função membro `get_future()` retorna o aprimorado [`std::experimental::future`](<#/doc/experimental/future>) fornecido por aquele TS.
