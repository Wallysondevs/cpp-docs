# std::experimental::packaged_task

`std::experimental::packaged_task` pode se referir a:   
  
  * [`std::experimental::fundamentals_v1::packaged_task` ou `std::experimental::fundamentals_v2::packaged_task`](<#/doc/experimental/lib_extensions/packaged_task>), uma versão modificada de [std::packaged_task](<#/doc/thread/packaged_task>) fornecida pelo [library fundamentals TS](<#/doc/experimental/memory>) com suporte para alocadores com type-erasure; 
  * [`std::experimental::concurrency_v1::packaged_task`](<#/doc/experimental/concurrency/packaged_task>), uma versão modificada de `std::packaged_task` fornecida pelo [concurrency TS](<#/doc/experimental/concurrency>) cuja função membro `get_future()` retorna o aprimorado [`std::experimental::future`](<#/doc/experimental/future>) fornecido por essa TS.
