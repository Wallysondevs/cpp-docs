# std::student_t_distribution&lt;RealType&gt;::n

RealType n() const; |  |  (desde C++11)  

  
Retorna o parâmetro de distribuição n (número de graus de liberdade) com o qual a distribuição foi construída. O valor padrão é 1.0. 

### Parâmetros

(nenhum) 

### Valor de retorno

Valor de ponto flutuante que identifica os graus de liberdade da distribuição.
