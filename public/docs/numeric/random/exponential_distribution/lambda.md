# std::exponential_distribution&lt;RealType&gt;::lambda

RealType lambda() const; |  |  (desde C++11)  

  
Retorna o parâmetro de distribuição λ com o qual a distribuição foi construída. O parâmetro define a taxa de eventos, por unidade. O valor padrão é 1.0. 

### Parameters

(none) 

### Return value

Valor de ponto flutuante que identifica a taxa de eventos por unidade.
