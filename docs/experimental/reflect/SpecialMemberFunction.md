# std::experimental::reflect::SpecialMemberFunction

Definido no cabeçalho `[<experimental/reflect>](<#/doc/header/experimental/reflect>)`

```c
template< class T >
concept SpecialMemberFunction = RecordMember<T> && /* see below */;
```

O concept `SpecialMemberFunction` é satisfeito se e somente se `T` reflete uma [função membro especial](<#/doc/language/member_functions>).
