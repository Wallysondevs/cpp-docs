# std::basic_regex&lt;CharT,Traits&gt;::flags

flag_type flags() const; | | (desde C++11)

Retorna as flags de sintaxe da expressão regular conforme definidas no construtor ou na última chamada para [assign()](<#/doc/regex/basic_regex/assign>).

### Parâmetros

(nenhum)

### Valor de retorno

Flags de sintaxe da expressão regular atuais.

### Exceções

Pode lançar exceções definidas pela implementação.
