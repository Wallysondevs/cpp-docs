# std::shuffle_order_engine&lt;Engine,K&gt;::max

```cpp
static constexpr result_type max();  // (desde C++11)
```

  
Retorna o valor máximo potencialmente gerado pelo adaptador de gerador. Este valor é igual a e.max() onde `e` é o gerador subjacente. 

### Parâmetros

(nenhum) 

### Valor de retorno

O valor máximo potencialmente gerado. 

### Complexidade

Constante.
