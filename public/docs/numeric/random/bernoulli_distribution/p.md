# std::bernoulli_distribution::p

double p() const; |  |  (desde C++11)  

  
Retorna o parâmetro p com o qual a distribuição foi construída. Ele define a probabilidade de gerar `true`. O valor padrão é 0.5. 

### Parâmetros

(nenhum) 

### Valor de retorno

Valor de ponto flutuante que identifica o parâmetro p da distribuição.
