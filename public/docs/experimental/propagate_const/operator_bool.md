# std::experimental::propagate_const&lt;T&gt;::operator bool

constexpr explicit operator bool() const; | | (library fundamentals TS v2)

Verifica se o objeto tipo ponteiro encapsulado, t_, é nulo.

### Parâmetros

(nenhum)

### Valor de retorno

true se o objeto encapsulado não for nulo, caso contrário false. Equivalente a (bool)t_.
