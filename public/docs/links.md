# Recursos úteis

### Standard C++ Foundation  
  
  * [isocpp.org](<https://isocpp.org/>) — Centro sem fins lucrativos para notícias, artigos e eventos de C++. 

    

  * [C++ FAQ](<https://isocpp.org/faq>) — FAQ unificado pela Standard C++ Foundation. 
  * [Mailing lists](<https://lists.isocpp.org/mailman/listinfo.cgi>) — As listas de e-mail públicas oficiais. 

### FAQs de C++

  * [Bjarne Stroustrup FAQ](<https://www.stroustrup.com/bs_faq.html>) — Respostas às perguntas frequentes do criador da linguagem. 
  * [Bjarne Stroustrup C++11 FAQ](<https://www.stroustrup.com/C++11FAQ.html>) — FAQ adicional sobre as mudanças recentes nos padrões da linguagem C++. 
  * [StackOverflow.com C++ FAQ](<https://stackoverflow.com/questions/tagged/c%2b%2b-faq?sort=votes>) — Melhores respostas para as perguntas frequentes de C++ no site de Q&A para programadores. 

### Documentos e rascunhos do padrão C++

  * [JTC1/SC22/WG21](<https://open-std.org/jtc1/sc22/wg21/>) — O Comitê de Padrões C++, inclui rascunhos de padrões da linguagem C++, propostas e relatórios de defeitos. 
    * N1146 ([PDF](<https://open-std.org/JTC1/SC22/WG21/docs/wp/pdf/nov97-2/>)) — Rascunho de trabalho final do C++98. 
    * N1804 ([PDF](<https://wg21.link/n1804>)) — Um rascunho de trabalho dos primeiros anos (2005-04-27). 
    * N3337 ([PDF](<https://wg21.link/n3337>), [HTML](<https://timsong-cpp.github.io/cppwp/n3337/>)) — Primeiro rascunho pós-publicação do C++11 (2012-01-16). 
    * N4140 ([PDF](<https://timsong-cpp.github.io/cppwp/n4140/draft.pdf>), [HTML](<https://timsong-cpp.github.io/cppwp/n4140/>)) — Rascunho de trabalho final do C++14 (2014-10-07). 
    * N4659 ([PDF](<https://wg21.link/n4659>), [HTML](<https://timsong-cpp.github.io/cppwp/n4659/>)) — Rascunho de trabalho final do C++17 (2017-03-21). 
    * N4861 ([PDF](<https://wg21.link/n4861>), [HTML](<https://timsong-cpp.github.io/cppwp/n4861/>)) — Rascunho de trabalho final do C++20 (2020-04-01). 
    * N4950 ([PDF](<https://wg21.link/n4950>), [HTML](<https://timsong-cpp.github.io/cppwp/n4950/>)) — Rascunho de trabalho final do C++23 (2023-05-10). 
    * N5001 ([PDF](<https://wg21.link/n5001>)) — Rascunho de trabalho mais recente do C++ (2024-12-17). 
  * [cplusplus/draft](<https://github.com/cplusplus/draft>) — O repositório oficial do GitHub do rascunho atual. 
  * [eel.is/c++draft](<https://eel.is/c++draft/>) — Versão HTML do rascunho atual. 

### Referências da linguagem e biblioteca C++

  * [Apache](<https://stdcxx.apache.org/doc/stdlibref/index.html>) — Manual de referência para a implementação Apache da biblioteca padrão C++. 
  * [IBM AIX 16.1](<https://www.ibm.com/docs/en/xl-c-and-cpp-aix/16.1>) [IBM zOS 3.1.0](<https://www.ibm.com/docs/en/zos/3.1.0>) — Manual de referência para a implementação IBM das linguagens C e C++ core. 
  * [IBM AIX 16.1](<https://www.ibm.com/docs/en/xl-c-and-cpp-aix/16.1?topic=reference-standard-c-library-header-files>) — Manual de referência para as implementações IBM da biblioteca padrão C++. 
  * [MSDN](<https://learn.microsoft.com/en-us/cpp/cpp/cpp-language-reference>) — Manual de referência para a implementação Microsoft da linguagem C++ core. 
  * [MSDN](<https://learn.microsoft.com/en-us/cpp/standard-library/cpp-standard-library-reference>) — Manual de referência para a implementação Microsoft da biblioteca C++. 
    * [Dinkumware](<https://web.archive.org/web/20120204111500/http://www.dinkumware.com/manuals/default.aspx>) — Implementação da Standard Library C++ que acompanha o Microsoft Visual C++. 
    * [Open STL](<https://github.com/microsoft/STL>) — Implementação da Standard Library C++ moderna do MSVC. 
  * [GNU libstdc++](<https://gcc.gnu.org/onlinedocs/libstdc++/index.html>) — Manual de referência para a implementação GNU da biblioteca C++. 
    * [libstdc++-v3](<https://github.com/gcc-mirror/gcc/tree/master/libstdc%2B%2B-v3>) — Implementação GNU da biblioteca C++. 
  * [libc++ documentation](<https://libcxx.llvm.org/>) — Documentação do projeto clang/llvm para a implementação da Standard Library C++. 
    * [libc++](<https://github.com/llvm/llvm-project/tree/main/libcxx>) — Implementação clang/llvm da Standard Library C++. 
  * [HP](<https://support.hpe.com/hpesc/public/docDisplay?docLocale=en_US&docId=emr_na-c02133105>) — Manual de referência para a implementação Dinkumware da biblioteca C++ (licenciada para a HP). 
  * [EWL](<https://www.nxp.com.cn/docs/en/reference-manual/CWEWLCPPREF.pdf>) — Manual de referência para a implementação Embedded Warrior da biblioteca C++. 
  * [cplusplus.com](<https://cplusplus.com/reference>) — Referência não oficial da biblioteca padrão C++. 
  * [SGI STL (hosted on rrsd.com)](<https://www.rrsd.com/software_development/stl/stl/index.html>) — Referência para STL, a biblioteca de 1994 que introduziu muitos conceitos posteriormente adotados pela biblioteca padrão C++.
