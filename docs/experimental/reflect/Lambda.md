# std::experimental::reflect::Lambda

Definido no cabeçalho `[<experimental/reflect>](<#/doc/header/experimental/reflect>)`

```c
template< class T >
concept Lambda = Type<T> && Scope<T> && /* see below */;
```

O conceito `Lambda` é satisfeito se e somente se T reflete um objeto closure (excluindo lambdas genéricas).
