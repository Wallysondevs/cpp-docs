# std::stop_source::stop_requested

```cpp
bool stop_requested() const noexcept;  // (desde C++20)
```

Verifica se o objeto `stop_source` possui um stop-state e se esse estado recebeu uma solicitação de parada.

### Parâmetros

(nenhum)

### Valor de retorno

true se o objeto `stop_token` possui um stop-state e se ele recebeu uma solicitação de parada, false caso contrário.
