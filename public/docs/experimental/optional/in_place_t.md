# std::experimental::in_place_t

Definido no cabeçalho `[<experimental/optional>](<#/doc/header/experimental/optional>)`

```c
struct in_place_t {};
```

`std::experimental::in_place_t` é um tipo de classe vazia usado para desambiguar as sobrecargas de construtores e funções membro que aceitam argumentos (possivelmente um parameter pack) para construção in-place de algum valor.
