# std::shuffle_order_engine&lt;Engine,K&gt;::min

```cpp
static constexpr result_type min();  // (desde C++11)
```

  
Retorna o valor mínimo potencialmente gerado pelo adaptador de engine. Este valor é igual a e.min() onde `e` é a engine subjacente. 

### Parâmetros

(nenhum) 

### Valor de retorno

O valor mínimo potencialmente gerado. 

### Complexidade

Constante.
