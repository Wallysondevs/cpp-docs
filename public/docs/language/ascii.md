Nota: no Unicode, o bloco de caracteres ASCII é conhecido como [`U+0000..U+007F` Latim Básico](<https://www.unicode.org/charts/PDF/U0000.pdf>).

### Exemplo

Execute este código
```cpp
    #include <iostream>
     
    int main()
    {
        std::cout << "Printable ASCII [32..126]:\n";
        for (char c{' '}; c <= '~'; ++c)
            std::cout << c << ((c + 1) % 32 ? ' ' : '\n');
        std::cout << '\n';
    }
```

Saída possível:
```

```
