# std::packaged_task&lt;R(Args...)&gt;::swap

```cpp
void swap( packaged_task& other ) noexcept;  // (desde C++11)
```

  
Troca os estados compartilhados e as tarefas armazenadas de *this e other. 

### Parâmetros

other  |  \-  |  packaged task cujo estado será trocado   
  
### Valor de retorno

(nenhum)
