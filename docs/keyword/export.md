# Palavra-chave C++: export

### Uso

Usado para marcar uma [definição de template](<#/doc/language/class_template>) como _exportada_, o que permite que o mesmo template seja declarado, mas não definido, em outras unidades de tradução. | (até C++11)
---|---
_A palavra-chave não é utilizada e é reservada._ | (desde C++11)
(até C++20)
Marca uma declaração, um grupo de declarações, ou outro [módulo](<#/doc/language/modules>) como exportado pelo [módulo](<#/doc/language/modules>) atual. | (desde C++20)
