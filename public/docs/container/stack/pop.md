# std::stack&lt;T,Container&gt;::pop

void pop();

  
Remove o elemento do topo da stack. Efetivamente chama c.pop_back(). 

### Parâmetros

(nenhum) 

### Valor de retorno

(nenhum) 

### Complexidade

Igual à complexidade de Container::pop_back.
