# std::function&lt;R(Args...)&gt;::~function

```cpp
~function();  // (desde C++11)
```

  
Destrói a instância de [std::function](<#/doc/utility/functional/function>). Se a `std::function` não estiver _vazia_, seu _alvo_ também é destruído.
