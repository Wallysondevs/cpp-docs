# std::experimental::nullopt

Definido no cabeçalho `[<experimental/optional>](<#/doc/header/experimental/optional>)`

```c
constexpr nullopt_t nullopt{/*unspecified*/};
```

  
`std::experimental::nullopt` é uma constante do tipo `std::experimental::nullopt_t` que é usada para indicar um tipo `optional` com estado não inicializado.
