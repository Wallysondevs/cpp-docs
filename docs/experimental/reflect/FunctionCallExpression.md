# std::experimental::reflect::FunctionCallExpression

Definido no cabeçalho `[<experimental/reflect>](<#/doc/header/experimental/reflect>)`

```c
template< class T >
concept FunctionCallExpression = Expression<T> && /* see below */;
```

  
O `concept` `FunctionCallExpression` é satisfeito se e somente se T reflete uma expressão de chamada de função.
