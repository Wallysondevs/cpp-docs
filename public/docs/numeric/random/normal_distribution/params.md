# std::normal_distribution&lt;RealType&gt;::mean, stddev

RealType mean() const; |  (1)  |  (desde C++11)  
---|---|---
RealType stddev() const; |  (2)  |  (desde C++11)  

  
Retorna os parâmetros com os quais a distribuição foi construída.

1) Retorna o parâmetro de distribuição média μ. A média especifica a localização do pico. O valor padrão é 0.0.

2) Retorna o parâmetro de distribuição desvio σ. O valor padrão é 1.0.

### Parâmetros

(nenhum) 

### Valor de retorno

1) O parâmetro de distribuição média μ.

2) O parâmetro de distribuição desvio σ.
