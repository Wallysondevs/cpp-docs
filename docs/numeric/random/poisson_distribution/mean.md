# std::poisson_distribution&lt;IntType&gt;::mean

double mean() const; |  |  (desde C++11)  

  
Retorna o parâmetro μ com o qual a distribuição foi construída. O parâmetro define o número médio de ocorrências do evento. O valor padrão é 1.0. 

### Parâmetros

(nenhum) 

### Valor de retorno

Valor de ponto flutuante que identifica o parâmetro μ da distribuição.
