# std::experimental::basic_string_view&lt;CharT,Traits&gt;::max_size

constexpr size_type max_size() const noexcept; |  |  (library fundamentals TS)  

  
O maior número possível de objetos tipo char que podem ser referenciados por um `basic_string_view`. 

### Parâmetros

(nenhum) 

### Valor de retorno

Número máximo de caracteres. 

### Complexidade

Constante.
