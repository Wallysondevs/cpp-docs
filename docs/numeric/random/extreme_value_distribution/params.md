# std::extreme_value_distribution&lt;RealType&gt;::a, b

RealType a() const; | (1) | (desde C++11)
---|---|---
RealType b() const; | (2) | (desde C++11)

Retorna os parâmetros com os quais a distribuição foi construída.

1) Retorna o parâmetro de distribuição `a` (localização). O valor padrão é 1.0.

2) Retorna o parâmetro de distribuição `b` (escala). O valor padrão é 0.0.

### Parâmetros

(nenhum)

### Valor de retorno

1) O parâmetro de distribuição `a` (localização).

2) O parâmetro de distribuição `b` (escala).
