# Cabeçalho da biblioteca padrão &lt;version&gt; (C++20)

Este cabeçalho faz parte da biblioteca de [suporte à linguagem](<#/doc/utility>).

Este cabeçalho fornece informações dependentes da implementação sobre a biblioteca padrão (como macros de versão da biblioteca específicas da implementação).

A inclusão de `< version>` também define todas as [macros de teste de recursos da biblioteca](<#/doc/feature_test>).

### Notas

Antes de C++20, a inclusão de [`<ciso646>`](<#/doc/header/ciso646>) era às vezes usada para este propósito.
