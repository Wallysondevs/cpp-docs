# std::chrono::leap_second::date

constexpr [std::chrono::sys_seconds](<#/doc/chrono/system_clock>) date() const noexcept;

  
Obtém a data e hora da inserção do segundo bissexto. 

### Valor de retorno

Um `std::chrono::sys_seconds` representando a data e hora em que o segundo bissexto foi inserido.
