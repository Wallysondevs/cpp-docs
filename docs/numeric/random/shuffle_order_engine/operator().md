# std::shuffle_order_engine&lt;Engine,K&gt;::operator()

```cpp
result_type operator()();  // (desde C++11)
```

  
Gera um valor aleatório. O estado do engine subjacente é avançado uma ou mais vezes. 

### Parâmetros

(nenhum) 

### Valor de retorno

Um número pseudoaleatório em [min(), max()]. 

### Exceções

Não lança exceções.
