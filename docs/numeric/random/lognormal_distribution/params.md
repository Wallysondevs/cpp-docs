# std::lognormal_distribution&lt;RealType&gt;::m, s

RealType m() const; |  (1)  |  (desde C++11)  
---|---|---
RealType s() const; |  (2)  |  (desde C++11)  

  
Retorna os parâmetros com os quais a distribuição foi construída.

1) Retorna o parâmetro de distribuição log-média m. Ele define a localização do pico. O valor padrão é 0.0.

2) Retorna o parâmetro de distribuição log-desvio s. O valor padrão é 1.0.

### Parâmetros

(nenhum) 

### Valor de retorno

1) O parâmetro de distribuição log-média m. 

2) O parâmetro de distribuição log-desvio s.
