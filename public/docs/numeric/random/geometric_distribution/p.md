# std::geometric_distribution&lt;IntType&gt;::p

double p() const; |  |  (desde C++11)  

  
Retorna o parâmetro de distribuição p com o qual a distribuição foi construída. O parâmetro define a probabilidade de uma tentativa gerar verdadeiro. O valor padrão é 0.5. 

### Parameters

(none) 

### Valor de retorno

Valor de ponto flutuante que identifica o parâmetro de distribuição p.
