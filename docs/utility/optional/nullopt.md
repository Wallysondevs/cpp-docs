# std::nullopt

Definido no cabeçalho `[<optional>](<#/doc/header/optional>)`

```c
inline constexpr nullopt_t nullopt{/*unspecified*/};
```

`std::nullopt` é uma constante do tipo [std::nullopt_t](<#/doc/utility/optional/nullopt_t>). Usado para indicar que um `std::optional` não contém um valor.
