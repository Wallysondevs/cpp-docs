# std::mdspan&lt;T,Extents,LayoutPolicy,AccessorPolicy&gt;::empty

```cpp
constexpr bool empty() const noexcept;  // (desde C++23)
```

  
Verifica se o [`mdspan`](<#/doc/container/mdspan>) está vazio. 

### Parâmetros

(nenhum) 

### Valor de retorno

true se o tamanho do espaço de índice multidimensional extents() for ​0​, caso contrário false.
