# std::discard_block_engine&lt;Engine,P,R&gt;::min

```cpp
static constexpr result_type min();  // (desde C++11)
```

  
Retorna o valor mínimo potencialmente gerado pelo adaptador de motor. Este valor é igual a e.min() onde `e` é o motor subjacente. 

### Parâmetros

(nenhum) 

### Valor de retorno

O valor mínimo potencialmente gerado. 

### Complexidade

Constante.
