# std::experimental::basic_string_view&lt;CharT,Traits&gt;::back

constexpr const_reference back() const; |  |  (library fundamentals TS)  

  
Retorna uma referência para o último caractere na view.

### Parâmetros

(nenhum)

### Valor de retorno

Referência para o último caractere, equivalente a operator[](size() - 1).

### Complexidade

Constante.
