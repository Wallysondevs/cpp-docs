# std::stacktrace_entry::operator=

```cpp
constexpr stacktrace_entry& operator=( const stacktrace_entry& other ) noexcept;  // (desde C++23)
```

  
Operador de atribuição por cópia. Substitui o conteúdo de *this pelo de `other`. 

### Parâmetros

other  |  \-  |  outra `stacktrace_entry` para atribuir   
  
### Valor de retorno

*this.
