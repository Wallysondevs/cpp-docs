# std::fisher_f_distribution&lt;RealType&gt;::m, n

RealType m() const; |  (1)  |  (desde C++11)  
---|---|---
RealType n() const; |  (2)  |  (desde C++11)  

  
Retorna os parâmetros com os quais a distribuição foi construída. 

1) Retorna o parâmetro de distribuição m (o primeiro grau de liberdade). O valor padrão é 1.0.

2) Retorna o parâmetro de distribuição n (o segundo grau de liberdade). O valor padrão é 1.0.

### Parâmetros

(nenhum) 

### Valor de retorno

1) O parâmetro de distribuição m (o primeiro grau de liberdade).

2) O parâmetro de distribuição n (o segundo grau de liberdade). 

### Complexidade

Constante.
