# std::filesystem::path::swap

```cpp
void swap( path& other ) noexcept;  // (1) (desde C++17)
```

  
Troca o conteúdo (tanto no formato nativo quanto genérico) de *this e `other`. 

### Parâmetros

other  |  \-  |  outro path para trocar o conteúdo   
  
### Valor de retorno

(nenhum) 

### Complexidade

Constante.
