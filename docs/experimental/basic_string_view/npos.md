# std::experimental::basic_string_view&lt;CharT,Traits&gt;::npos

static constexpr size_type npos = size_type(-1); |  |  (library fundamentals TS)  

  
Este é um valor especial igual ao valor máximo representável pelo tipo `size_type`. O significado exato depende do contexto, mas é geralmente usado como um indicador de fim de view pelas funções que esperam um índice de view ou como um indicador de erro pelas funções que retornam um índice de view.
