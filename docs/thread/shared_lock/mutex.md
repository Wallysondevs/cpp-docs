# std::shared_lock&lt;Mutex&gt;::mutex

```cpp
mutex_type* mutex() const noexcept;  // (desde C++14)
```

  
Retorna um ponteiro para o mutex associado, ou um ponteiro nulo se não houver mutex associado. 

### Parâmetros

(nenhum) 

### Valor de retorno

Ponteiro para o mutex associado ou um ponteiro nulo se não houver mutex associado.
