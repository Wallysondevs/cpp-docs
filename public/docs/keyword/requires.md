# Palavra-chave C++: requires (desde C++20)

### Uso

  * especifica uma expressão constante em parâmetros de template que [avaliam um requisito](<#/doc/language/requires>)

  * dentro de tal requisito, introduz um [requisito aninhado](<#/doc/language/requires>)

  * em uma declaração de template, especifica uma [restrição associada](<#/doc/language/constraints>)
