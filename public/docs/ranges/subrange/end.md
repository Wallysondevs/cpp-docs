# std::ranges::subrange&lt;I,S,K&gt;::end

```cpp
constexpr S end() const;  // (desde C++20)
```

  
Retorna o sentinel que indica o fim do [`subrange`](<#/doc/ranges/subrange>). 

### Valor de retorno

`_[end_](<#/doc/ranges/subrange>)_`. 

### Exemplo

| Esta seção está incompleta  
Razão: nenhum exemplo
