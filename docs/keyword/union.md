# Palavra-chave C++: union

### Uso

  * [ declaração de um tipo union](<#/doc/language/union>)
  * Se uma função ou uma variável existir no escopo com o nome idêntico ao nome de um tipo union, `union` pode ser prefixado ao nome para desambiguação, resultando em um [especificador de tipo elaborado](<#/doc/language/elaborated_type_specifier>)
