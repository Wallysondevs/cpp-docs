# C++ keyword: typename

### Uso

  * Na lista de parâmetros de template de uma [declaração de template](<#/doc/language/templates>), `typename` pode ser usado como uma alternativa a [class](<#/doc/keyword/class>) para declarar [parâmetros de template de tipo](<#/doc/language/template_parameters>) e parâmetros de template de template (desde C++17).
  * Dentro de uma declaração ou definição de um template, `typename` pode ser usado para declarar que um [nome qualificado dependente](<#/doc/language/dependent_name>) é um tipo.
  * `typename` pode ser usado antes de um nome de tipo qualificado não dependente. Não tem efeito neste caso.

  * Dentro de um [requirements](<#/doc/language/constraints>) para requisitos de tipo

| (desde C++20)
