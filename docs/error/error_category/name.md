# std::error_category::name

```cpp
virtual const char* name() const noexcept = 0;  // (desde C++11)
```

  
Retorna um ponteiro para uma string de bytes terminada em nulo que especifica o nome da categoria de erro. 

### Parâmetros

(nenhum) 

### Valor de retorno

String de bytes terminada em nulo especificando o nome da categoria de erro.
